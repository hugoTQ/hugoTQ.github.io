## 1.怎样 定义一个结构体数组

1. 使用指针

   ```c
   // 分配内存
   A *a = (A*)calloc(sizeof(A), 10);
   printf("size: %d\n", sizeof(a));
   // 清零
   
   // 释放
   free(a);
   
   
   ```

2. 使用指针的指针

   ``` c
   A **a =(A**)calloc(sizeof(A), 10);
   printf("size: %d\n", sizeof(a));
   free(a);
   ```

3. 使用数组指针

   ```c
   A a[10] = (A*)calloc(sizeof(A), 10);
   printf("size: %d\n", sizeof(a));
   free(a);
   ```
