```c
void main(){
        char a[256] = {0};
        printf("a=[%ld], &a=[%ld]\n", a, &a);
        a[0] = 'h';
        printf("[%s]\n", a);
        memset(&a, 0, sizeof(a));
        printf("[%s]\n", a);
        memset(a, 0, sizeof(a));
        printf("[%s]\n", a);
}

output:
a=[140730453755376], &a=[140730453755376]
[h]
[]
[]
```

上面说明数组a是第一个元素地址，&a也是第一个元素地址
